# Zanzibar Fresh OS

A React/Vite prototype for managing a small hydroponic produce business supplying boutique hotels in Zanzibar.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repo into Vercel.
3. Framework preset: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.

## Current modules

- Dashboard KPIs
- Revenue and profit charts
- NFT channel farm map
- Crop/product planner
- Hotel customer list
- Delivery rhythm
- Financial controls
- pH/EC nutrient logging
